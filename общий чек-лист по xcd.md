### Иностранная организация/гражданин (`СвИнНеУч`)
- [ ] 7.5.1 Заполнен хотя бы один из атрибутов: `Идентиф` или `ИныеСвед`

## Блок 8. Подписант (`ПодпИнф*`) — Универсальная логика для всех КНД

### Основные атрибуты
- [ ] 8.1 Блок `ФИО` **обязателен**: `Фамилия` и `Имя` (1–60 символов), `Отчество` опционально (1–60)
- [ ] 8.2 `Должн` **обязателен** (длина от 1 до 255)
- [ ] 8.3 `СпосПодтПолном` **обязателен** (значение из списка: `1`, `2`, `3`, `4`, `5`, `6`)

### Логика подтверждения полномочий
**Если `СпосПодтПолном` = `"3"` (Электронная доверенность / МЧД):**
- [ ] 8.4.1 Блок `СвДоверЭл` **заполнен**
- [ ] 8.4.2 Блок `СвДоверБум` **отсутствует**
- [ ] 8.4.3 Внутри `СвДоверЭл`: `НомДовер` (36 символов), `ДатаВыдДовер` (дата), `ИдСистХран` (1–500) **обязательны**
- [ ] 8.4.4 Необязательные атрибуты `СвДоверЭл`: `ВнНомДовер` (1–50), `ДатаВнРегДовер` (дата), `УРЛСист` (1–500)

**Если `СпосПодтПолном` = `"5"` (Бумажная доверенность):**
- [ ] 8.5.1 Блок `СвДоверБум` **заполнен**
- [ ] 8.5.2 Блок `СвДоверЭл` **отсутствует**
- [ ] 8.5.3 Внутри `СвДоверБум`: `ДатаДовер` (дата) и `ВнНомДовер` (1–50) **обязательны**
- [ ] 8.5.4 Необязательные атрибуты `СвДоверБум`: `СвИдДовер` (1–1000), блок `ФИО`

**Если `СпосПодтПолном` = 1, 2, 4, 6:**
- [ ] 8.6.1 Блоки `СвДоверЭл` и `СвДоверБум` **отсутствуют**

### Необязательные атрибуты подписанта
- [ ] 8.7 `ТипПодпис` (если есть) — значение из списка: `1`, `2`, `3`
- [ ] 8.8 `ДатаПодДок` (если есть) — формат `ДД.ММ.ГГГГ`
- [ ] 8.9 `ДопСведПодп` (если есть) — длина от 1 до 255 символов

## Блок 9. Проверки типов данных — Общая для всех КНД

- [ ] 9.1 **Дата (`ДатаТип`)**: формат `ДД.ММ.ГГГГ`, реальная календарная дата
- [ ] 9.2 **Время (`ВремяТип`)**: формат `ЧЧ:ММ:СС`
- [ ] 9.3 **ИНН ЮЛ**: 10 цифр, маска `([0-9]{1}[1-9]{1}|[1-9]{1}[0-9]{1})[0-9]{8}`
- [ ] 9.4 **ИНН ФЛ/ИП**: 12 цифр, маска `([0-9]{1}[1-9]{1}|[1-9]{1}[0-9]{1})[0-9]{10}`
- [ ] 9.5 **КПП**: 9 знаков
- [ ] 9.6 **ОГРН/ОГРНИП**: 13 / 15 цифр
- [ ] 9.7 **БИК**: 9 цифр
- [ ] 9.8 **Код страны (ОКСМ)**: 3 цифры
- [ ] 9.9 **Код региона (ССРФ)**: 2 цифры
- [ ] 9.10 **Почтовый индекс**: 6 цифр
- [ ] 9.11 **GUID** (`ИдНом`, `НомДовер`, `ИдФайлИнфПол`): 36 символов
- [ ] 9.12 **GLN** (если указан): 13 цифр
- [ ] 9.13 **СумНДС (`СумНДСТип`)**: выбор ровно одного варианта:
    - `СумНал` — число (до 19 знаков, 2 знака после запятой, ≥ 0)
    - `БезНДС` — строка `"без НДС"`

- [ ] 10.4 `РеквДокТип` (если используется): атрибуты `РеквНаимДок`, `РеквНомерДок`, `РеквДатаДок` **обязательны**
