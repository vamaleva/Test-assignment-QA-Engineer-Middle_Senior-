# Решения SQL-задач (PostgreSQL)

## 1. Вторая зарплата

**Таблица:** `Employee (id INT PK, salary INT)`  
**Задача:** Найти вторую по величине уникальную зарплату. Если её нет — вернуть `NULL`.

### Основной запрос

SELECT (
    SELECT DISTINCT salary
    FROM Employee
    ORDER BY salary DESC
    LIMIT 1 OFFSET 1
) AS "SecondHighestSalary";

### Альтернативный вариант с DENSE_RANK

SELECT MAX(salary) AS "SecondHighestSalary"
FROM (
    SELECT salary, DENSE_RANK() OVER (ORDER BY salary DESC) AS rnk
    FROM Employee
) ranked
WHERE rnk = 2;

## 2. Поиск дубликатов email

**Таблица:** `Person (id INT PK, email VARCHAR)`  
**Задача:** Найти все email-адреса, которые встречаются более одного раза.

### Запрос

SELECT email AS "Email"
FROM Person
GROUP BY email
HAVING COUNT(*) > 1;

## 3. Клиенты без заказов

**Таблицы:** `Customers (id INT PK, name VARCHAR)` и `Orders (id INT PK, customerId INT FK)`  
**Задача:** Вернуть имена клиентов, которые ни разу не делали заказов.

### Вариант с LEFT JOIN

SELECT c.name AS "Customers"
FROM Customers c
LEFT JOIN Orders o ON c.id = o.customerId
WHERE o.id IS NULL;

### Вариант с NOT EXISTS (рекомендуется)

SELECT c.name AS "Customers"
FROM Customers c
WHERE NOT EXISTS (
    SELECT 1
    FROM Orders o
    WHERE o.customerId = c.id
);

### Вариант с NOT IN

SELECT name AS "Customers"
FROM Customers
WHERE id NOT IN (
    SELECT DISTINCT customerId
    FROM Orders
    WHERE customerId IS NOT NULL
);
